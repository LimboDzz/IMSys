## 安装

这个项目使用 [node](http://nodejs.org) 和 [npm](https://npmjs.com)。使用之前要先在本地安装好。

```sh
$ IMSys> cd server
$ IMSys\server> npm install
$ IMSys\server\src> node .\app.js

$ IMSys> cd client
$ IMSys\client> npm install
$ IMSys\client> npm run dev
```
